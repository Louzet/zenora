# Zenora Docs

Concise walkthroughs to explore Zenora features. Each guide mixes explanation and copy-paste ready snippets you can drop into your own commands.

## Contents
- `docs/tasks.md`: tasks, spinner, progress bar
- `docs/tables.md`: styled tables, alignment, wrapping
- `docs/services.md`: custom services, before/after hooks
- `docs/select.md`: interactive select with non-TTY fallback

Copy/paste the snippets into your own commands or adapt them for local demos and screenshots.
